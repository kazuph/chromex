Chromex Privacy Policy

Effective date: April 28, 2026

Chromex is a Chrome side-panel assistant that helps users work with webpages, selected tabs, uploaded files, voice input, images, and browser workflows through a local Codex bridge.

This Privacy Policy explains what data Chromex may process, why it is processed, and how the extension is designed to limit access to user-requested tasks.

What Chromex Does

Chromex can summarize webpages, compare selected tabs, answer questions about page content, draft text, process uploaded files, work with images, use voice input, search browser history when requested, and help with browser workflows.

Chromex is designed around user-controlled context. Page content, tab data, screenshots, history, files, images, audio, and browser actions are used only when a user requests a feature that needs that information.

Data Chromex May Process

  • Website content, including page text, selected text, DOM-derived content, images, videos, links, screenshots, and visible or readable content from the current page.
  • Tab information, including titles, URLs, favicons, and user-selected open tabs.
  • Uploaded files and images selected by the user.
  • Voice audio and transcripts when the user uses voice input or live voice features.
  • Browser history only when the user asks history-based questions and grants the required Chrome permission.
  • User activity needed for requested browser workflows, such as clicks, scrolling, DOM-assisted actions, and visible page interaction planning.
  • Personal communications or personally identifiable information only when such data appears in user-selected content, uploaded files, communications, or browser history.

Chromex does not intentionally collect health information, financial or payment information, precise location data, or authentication secrets as part of its single purpose.

Authentication and Local Bridge

Chromex uses a local native bridge to communicate with the user's configured Codex/OpenAI runtime.

The Chrome extension does not store raw OpenAI API keys, OAuth tokens, or ChatGPT session tokens in Chrome extension storage. Authentication handoff, API-key fallback, generated-image files, diagnostics, and runtime-heavy operations are handled by the local bridge installed on the user's computer.

Use, Sharing, and Storage

Chromex uses data only to provide the feature requested by the user. Data may be sent through the local bridge to the user's configured Codex/OpenAI service when needed to complete that task.

Chromex does not sell user data.

Chromex does not use user data for advertising and does not use user data to determine creditworthiness or for lending purposes.

Chromex may store local extension preferences such as theme, language, selected model, profile templates, enabled skills, onboarding state, and lightweight conversation metadata. Large generated assets, temporary files, and diagnostics are handled by the local bridge rather than Chrome extension storage.

Permissions and User Control

Chromex requests baseline permissions for the side-panel assistant and requests optional permissions only when a feature needs them. Users control when Chromex receives page context, selected tabs, uploaded files, images, voice input, browser history, and browser action permissions. Users can disable optional features, revoke Chrome permissions, clear local settings, and remove generated files.

Remote Code

Chromex does not execute remotely hosted JavaScript or remotely supplied extension code. Extension code is packaged with the MV3 extension.

Contact

For privacy or support questions, contact support@genexis.ai. If this address is not yet active, use the GitHub issue tracker.

Chromex 개인정보 처리방침

시행일: 2026년 4월 28일

Chromex는 로컬 Codex 브리지를 통해 웹페이지, 선택한 탭, 업로드 파일, 음성 입력, 이미지, 브라우저 작업을 처리하는 Chrome 사이드패널 어시스턴트입니다.

이 방침은 Chromex가 어떤 데이터를 처리할 수 있는지, 왜 처리하는지, 그리고 사용자가 요청한 작업 범위로 접근을 제한하기 위해 어떻게 설계되었는지 설명합니다.

Chromex가 하는 일

Chromex는 웹페이지 요약, 선택 탭 비교, 페이지 내용 질문 답변, 글 초안 작성, 업로드 파일 처리, 이미지 작업, 음성 입력, 사용자가 요청한 브라우저 기록 검색, 브라우저 작업 보조를 수행할 수 있습니다.

Chromex는 사용자가 선택한 문맥을 중심으로 작동합니다. 페이지 내용, 탭 정보, 스크린샷, 기록, 파일, 이미지, 오디오, 브라우저 동작은 해당 기능을 사용자가 요청한 경우에만 사용됩니다.

처리될 수 있는 데이터

  • 현재 페이지의 텍스트, 선택 텍스트, DOM 기반 내용, 이미지, 영상, 링크, 스크린샷 등 웹사이트 내용
  • 탭 제목, URL, favicon, 사용자가 선택한 열린 탭 정보
  • 사용자가 업로드하거나 선택한 파일과 이미지
  • 음성 입력 또는 라이브 음성 기능 사용 시 마이크 오디오와 전사 텍스트
  • 사용자가 브라우저 기록 기반 질문을 요청하고 권한을 허용한 경우의 방문 기록
  • 클릭, 스크롤, DOM 보조 작업, 페이지 상호작용 계획 등 사용자가 요청한 브라우저 작업에 필요한 활동 정보
  • 사용자가 선택한 콘텐츠, 업로드 파일, 커뮤니케이션, 브라우저 기록 안에 포함된 경우의 개인 식별 정보 또는 개인 커뮤니케이션 내용

Chromex는 단일 목적 수행을 위해 건강 정보, 금융 또는 결제 정보, 정밀 위치 정보, 인증 비밀을 의도적으로 수집하지 않습니다.

인증과 로컬 브리지

Chromex는 사용자가 설정한 Codex/OpenAI 런타임과 통신하기 위해 로컬 네이티브 브리지를 사용합니다.

Chrome 확장 프로그램 저장소에는 원본 OpenAI API 키, OAuth 토큰, ChatGPT 세션 토큰을 저장하지 않습니다. 인증 전달, API 키 폴백, 생성 이미지 파일, 진단, 무거운 런타임 작업은 사용자의 컴퓨터에 설치된 로컬 브리지가 처리합니다.

사용, 공유, 저장

Chromex는 사용자가 요청한 기능을 제공하기 위해서만 데이터를 사용합니다. 작업 수행에 필요한 경우 로컬 브리지를 통해 사용자가 설정한 Codex/OpenAI 서비스로 데이터가 전송될 수 있습니다.

Chromex는 사용자 데이터를 판매하지 않고, 광고에 사용하지 않으며, 신용 평가나 대출 목적으로 사용하지 않습니다.

Chromex는 테마, 언어, 선택 모델, 프로필 템플릿, 활성화된 스킬, 온보딩 상태, 가벼운 대화 메타데이터 같은 로컬 설정을 저장할 수 있습니다. 큰 생성 자산, 임시 파일, 진단 정보는 Chrome 확장 프로그램 저장소가 아니라 로컬 브리지가 처리합니다.

권한과 사용자 제어

Chromex는 사이드패널 어시스턴트에 필요한 기본 권한을 요청하고, 특정 기능에 필요할 때만 선택 권한을 요청합니다. 사용자는 페이지 문맥, 선택한 탭, 업로드 파일, 이미지, 음성 입력, 브라우저 기록, 브라우저 작업 권한 제공 시점을 제어할 수 있습니다.

원격 코드

Chromex는 원격 호스팅 JavaScript 또는 원격에서 제공된 확장 코드를 실행하지 않습니다. 확장 코드는 MV3 확장 패키지에 포함되어 배포됩니다.

Chromex プライバシーポリシー

施行日: 2026 年 4 月 28 日

Chromex は、ローカル Codex ブリッジを通じて、ウェブページ、選択したタブ、アップロードファイル、音声入力、画像、ブラウザ操作を扱う Chrome サイドパネルアシスタントです。

このポリシーは、Chromex が処理する可能性のあるデータ、その目的、そしてユーザーが要求した作業にアクセスを限定する設計を説明します。

Chromex の機能

Chromex は、ページ要約、タブ比較、ページ内容への質問回答、文章作成、アップロードファイル処理、画像作業、音声入力、ユーザーが求めたブラウザ履歴検索、ブラウザ操作支援を行えます。

処理される可能性のあるデータ

  • ページ本文、選択テキスト、DOM 由来の内容、画像、動画、リンク、スクリーンショットなどのウェブサイト内容
  • タブのタイトル、URL、favicon、ユーザーが選択した開いているタブ
  • ユーザーがアップロードまたは選択したファイルと画像
  • 音声入力やライブ音声機能を使う場合の音声と文字起こし
  • ユーザーが履歴検索を求め、Chrome 権限を許可した場合のブラウザ履歴
  • クリック、スクロール、DOM 補助操作など、ユーザーが要求したブラウザワークフローに必要な操作情報
  • 選択された内容やアップロードファイルに含まれる場合の個人情報やコミュニケーション内容

Chromex は、単一目的の一部として健康情報、金融・支払い情報、正確な位置情報、認証秘密を意図的に収集しません。

認証とローカルブリッジ

Chromex は、ユーザーが設定した Codex/OpenAI ランタイムと通信するためにローカルネイティブブリッジを使用します。Chrome 拡張機能ストレージには、生の OpenAI API キー、OAuth トークン、ChatGPT セッショントークンを保存しません。

利用、共有、保存

Chromex は、ユーザーが要求した機能を提供するためだけにデータを使用します。必要な場合、ローカルブリッジを通じてユーザーが設定した Codex/OpenAI サービスにデータが送信されることがあります。

Chromex はユーザーデータを販売せず、広告に使用せず、信用判断や融資目的にも使用しません。

Chromex はテーマ、言語、選択モデル、プロファイルテンプレート、有効なスキル、オンボーディング状態、軽量な会話メタデータなどのローカル設定を保存することがあります。

権限とユーザー制御

Chromex はサイドパネルアシスタントに必要な基本権限を要求し、機能が必要とする場合だけ任意権限を要求します。ユーザーは、ページ文脈、選択タブ、アップロードファイル、画像、音声入力、履歴、ブラウザ操作権限をいつ渡すか制御できます。

リモートコード

Chromex はリモートでホストされた JavaScript や、リモートから供給された拡張コードを実行しません。拡張コードは MV3 拡張パッケージに含まれます。

連絡先

プライバシーまたはサポートに関する質問は support@genexis.ai までご連絡ください。このアドレスが利用できない場合は GitHub issue tracker を利用できます。

Chromex 隐私政策

生效日期: 2026 年 4 月 28 日

Chromex 是一个 Chrome 侧边栏助手，通过本地 Codex bridge 帮助用户处理网页、选中的标签页、上传文件、语音输入、图片和浏览器工作流。

本政策说明 Chromex 可能处理哪些数据、处理原因，以及如何把访问范围限制在用户请求的任务内。

Chromex 的功能

Chromex 可以总结网页、比较选中的标签页、回答页面内容问题、起草文本、处理上传文件、处理图片、使用语音输入、在用户请求时搜索浏览器历史，并帮助完成浏览器工作流。

可能处理的数据

  • 网页文本、选中文本、DOM 提取内容、图片、视频、链接、截图等网站内容
  • 标签页标题、URL、favicon 和用户选择的打开标签页
  • 用户上传或选择的文件和图片
  • 用户使用语音输入或实时语音功能时的音频和转写文本
  • 用户请求基于历史记录的问题并授予 Chrome 权限时的浏览器历史
  • 完成用户请求的浏览器工作流所需的点击、滚动、DOM 辅助操作和页面交互信息
  • 当用户选择的内容、上传文件、通信内容或浏览器历史中包含时的个人信息或个人通信内容

Chromex 不会为了其单一用途而有意收集健康信息、金融或支付信息、精确位置数据或认证密钥。

认证与本地 Bridge

Chromex 使用本地 native bridge 与用户配置的 Codex/OpenAI 运行时通信。Chrome 扩展存储不会保存原始 OpenAI API key、OAuth token 或 ChatGPT session token。

使用、共享与存储

Chromex 仅为提供用户请求的功能而使用数据。必要时，数据可能通过本地 bridge 发送到用户配置的 Codex/OpenAI 服务。

Chromex 不出售用户数据，不将用户数据用于广告，也不用于信用评估或贷款目的。

Chromex 可能保存主题、语言、所选模型、配置文件模板、启用的 skills、引导状态和轻量会话元数据等本地偏好设置。

权限与用户控制

Chromex 请求侧边栏助手所需的基础权限，并仅在功能需要时请求可选权限。用户可以控制何时提供页面上下文、标签页、上传文件、图片、语音输入、浏览器历史和浏览器操作权限。

远程代码

Chromex 不执行远程托管的 JavaScript，也不执行远程提供的扩展代码。扩展代码随 MV3 扩展包一起发布。

联系

隐私或支持问题请联系 support@genexis.ai。如果该地址暂不可用，也可以使用 GitHub issue tracker。
